# 5x5MatrixLIB
for Infineon/SIEMENS/OSRAM Intelligent Display (5x5) ROMless series ONLY

Supported Device in listed down below (X mean color or LED,Y is Package)
=
SCD558X(A) series
SCD5510X   series
SCDV554X   series
SCE574X    series
SCE578X    series
SCDQ554XY  Series
